###### 一些术语

- 词法单元(token)/单词符号: 由一个词法单元名(单词种别)和一个可选的属性值组成

  (种别，属性)

- 词素(lexeme)：源程序中的一个字符序列，与某个词法单元的模式匹配，被识别为词法单元的一个实例

- 模式(pattern): 描述了一个此法单元的词素可能具有的形式

  | 词法单元名(token) | 模式(非正式描述)                      | 词素(lexemes)       |
  | ----------------- | ------------------------------------- | ------------------- |
  | if                | characters i, f                       | if                  |
  | else              | characters e, l, s, e                 | else                |
  | comparison        | < or > or <= or >= or == or !=        | <=, !=              |
  | id                | letter followed by letters and digits | pi, score, D2       |
  | number            | any numeric constant                  | 3.14159, 0, 6.02e23 |
  | literal           | anything but ", surrounded by "'s     | "core dumped"       |

  

- 标识符(identifier): 表示各种名字

- 关键字(keyword): 程序语⾔定义的具有固定意义的标识符

- 常数(constant): 包括整型，浮点型，布尔型

- 符号(symbols): 包括运算符+，*，++，<<，<，<=，界符，{，}，[，]等

- 注释(comments): /* bla bla bla */

###### 词法分析器的输出形式

###### 单词符号分类

- 关键字

  可将其全体视为一种，也可以一字一种，后者更方便

- 标识符

  可以是一种，也可以按类型分种，一般统归为一种

- 常数

  宜按照类型(整型、实型、布尔型等)分种

- 运算符

  可采用一字一种的分法

- 状态之间用箭弧连接

#### 有限自动机

有限自动机作为一种识别装置，能准确地识别正则集合

- 确定有限自动机(DFA)
- 不确定有限自动机(NFA)

###### 确定有限自动机

一个确定的有限自动机是一个五元式: $M=(S, \Sigma$, $\delta, s_0, F)$

- $S$是一个有限集，它的每一个元素称为一个状态

- $\Sigma$是一个有穷字母表，它的每个元素称为一个输入字符

- $\delta:{S}\times\Sigma\rightarrow{S}$是一个单值映射(确定性)

  $\delta(s,a)=s^\prime$意味着: 当现行状态为s、输入字符为a时，将转换到下一个状态s’

  我们称s’为s的一个后继状态

- $s_0\in{S}$是唯一的初态

- $F\subseteq S$是一个终态集(可空)

确定的有限自动机的表示方式:

- 状态转换矩阵

| State |   a   |   b   |
| :---: | :---: | :---: |
|   A   |   B   |   A   |
|   B   |   B   |   C   |
|   C   |   B   |   D   |
|   D   |   B   |   A   |

###### 非确定有限自动机

一个非确定有限自动机是一个五元式: $M=(S, \Sigma$, $\delta, S_0, F)$

- $S$是一个有限集，它的每一个元素称为一个状态
- $\Sigma$是一个有穷字母表，它的每个元素称为一个输入字符
- $\delta:{S}\times\Sigma^*\rightarrow2^{S}$是一个从$S\times\Sigma^*$到$S$子集的单值映射
- $S_0$是一个非空的初态集
- $F\subseteq S$是一个终态集(可空)

若$\delta$是一个多值函数，且输入可允许为$\epsilon$，则有限自动机是不确定的，也就是说在某个状态下对于某个输入字符有多个后继状态

非确定有限自动机的表示方式: 同确定有限自动机的两种方式

###### NFA的确定化

状态集合$I$的$\epsilon$-闭包，表示为$\epsilon-closure(I)$，定义为一状态集，包括

- 状态集合$I$的任何状态$s$都属于$\epsilon-closure(I)$
-  状态集合$I$中的任何状态$s$经任意条$\epsilon$弧而能到达的状态的集合

状态集合$I$的$a$弧转换$I_a=\epsilon-closure(move(I, a))=e-closure(J)$，其中$move(I, a)$为状态集合$J$，是所有那些可以从$I$的某一状态经过一条$a$弧而到达的状态的全体

**使用子集法进行NFA的确定化**

1. 对任何一个具有$\epsilon-$转移的NFA $M$，一定存在一个不具有$\epsilon-$转移 的 NFA $M’$，使$L(M’)=L(M)$
2. 对于字母表$S$上任何一个具有$\epsilon-$转移的NFA $M$，一定存在 一个的DFA $M’$，使得$L(M’) = L(M)$
3. 对任何一个NFA $M =(S, \Sigma, f, S_0, F)$，都存在一个DFA $M’ =(S’, S, f’, s_0, F’)$，使得$L(M’)=L(M)$

###### DFA的最小化

DFA的最小化是指一个有穷自动机通过消除多余状态和合并等价状态而转换成一个最小的与之等价的有穷自动机，即寻找一个状态数比$M$少的DFA $M'$，使得$L(M)=L(M')$

最小状态DFA:

- 不存在多余状态(死状态)
- 不存在互相等价(不可区别)的两个状态

对于两个状态S和T等价，就是说从S出发可以读出某个字符串w到达终态，从T出发也可以读出字符串w到达终态，反之亦然

两个状态S和T可区别是两个状态不满足下面两条:

- 兼容性(一致性): 同是终态或同是非终态
- 传播性(蔓延性): 从状态S出发读入某个$a(a\in\Sigma)$和从状态T出发读入某个$a$到达的状态等价

DFA $M =（S,\Sigma, \delta, s_0, S_t)$，最小状态DFA $M’$ 

1. 构造状态的初始划分$\Pi$: 终态$S_t$和非终态$S- S_t$两组
2. 对$\Pi$施用传播性原则构造新划分$\Pi_{new}$
3. 如$\Pi_{new} == \Pi$，则令$\Pi_{final}=\Pi$并继续步骤4，否则$\Pi := \Pi_{new}$重复2
4. 为$\Pi_{final}$中的每一组选一代表，这些代表构成$M’$的状态，若s是一代表，且$ \delta(s,a)=t$，令$r$是$t$组的代表，则$M’$中有一转换$\delta’(s, a)=r$，$M’$的开始状态是含有$s_0$的那组的代表，$M’$的终态是含有$s_t$的那组的代表
5. 去掉$M’$中的死状态
